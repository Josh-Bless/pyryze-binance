# pyryze-binance

A lightweight **Java reactive library** for interacting with the Binance API using Project Reactor.

## 🚀 Features
- Fully reactive (built on Project Reactor)
- Non-blocking Binance API integration
- Lightweight and dependency-minimal
- Designed for microservices and fintech systems

## 📦 Installation (Maven)

```xml
<dependency>
    <groupId>com.yourname</groupId>
    <artifactId>pyryze-binance</artifactId>
    <version>1.0.0</version>
</dependency>
